# App icon source

`AppIcon.png` is an original project-bound bitmap generated with OpenAI's built-in image-generation tool, then converted from a flat magenta chroma-key canvas to transparent corners with the installed image-generation skill helper.

Final generation prompt:

> Create an original, polished macOS icon for a calm productivity utility. Show a warm amber doorway or threshold of light held inside a deep pine-green rounded architectural form, communicating a deliberate pause before crossing into distraction. Use a premium minimal, vector-friendly 3D style with a bold small-size silhouette. No text, letters, numbers, people, clocks, locks, game controllers, copied branding, or watermark.

The project packaging script derives the required `.icns` representations from this PNG.
